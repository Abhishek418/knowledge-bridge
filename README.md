# Knowledge Bridge

A Spring Boot service leveraging **Spring AI** for AI-native development.

## Tech Stack

- **Java 21+**
- **Spring Boot 3.x**
- **Spring AI** - For seamless AI/LLM integrationß
- **Gradle** - Build tool



## Spring AI Features

This service is designed for AI-native development with Spring AI, supporting:

- **Chat Models** - Integration with various LLM providers (OpenAI, Anthropic, Ollama, etc.)
- **Embeddings** - Vector embeddings for semantic search
- **RAG (Retrieval Augmented Generation)** - Enhanced context-aware responses
- **Function Calling** - Tool/function execution with LLMs
- **Structured Output** - Type-safe AI responses
